Mooc Service

API Exposed Endpoints

GET /tags

A list of tags used to categorize mooc subject content. These tags are matched exactly.

GET /tags/{tag}

Enter a {tag} to see all moocs marked with that tag.

For example: /tags/Java

GET /moocs

A list of all moocs.

GET /moocs/{tags}

Enter /moocs/{tags} to see all moocs matching any {tags}.

For example: /moocs/Java,CSS will display all moocs tagged with Java or CSS.
